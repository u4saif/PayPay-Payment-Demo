# _PayPay-Payment-Demo_
PayPay Payment Demo in Angular 8.

In this Project we have implemented the frontend of PayPal checkout and hosted the site on Google firebase hosting.
This project shows how to capture the user's details and make a payment for the purchase product from their paypal accout or creditcard. 

# *You can check the project live here* : [ Click to see it](https://fire4pay.web.app/)

![](https://raw.githubusercontent.com/u4saif/PayPay-Payment-Demo/master/image.png)

Note: Command used for firebase Deployemnt are as follow. 

Firstly buld the project with **ng build --prod**
Now install the firebase tolls **npm i -g firebase-tools** in your machine to make use of Google fire base cli. **firebase login** **firebase list (to select project)**  

Initialize the directory firebase init make sure you change the path for project to **/dist/name_of_your_project** by default it would be public 

Lastly deploy the app using **firebase deploy**
